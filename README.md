IntentFlow is a full-stack platform (Web app + Chrome extension + Node backend) that captures live AI chat search intent, maps it into campaign-version trees, enriches results with SEO keyword intelligence (Semrush/Ahrefs), and supports refire-based versioning to compare campaign outcomes over time. It enables end-to-end workflow from onboarding and authentication to multi-provider prompt execution, chat-thread history, and structured dashboard insights.
